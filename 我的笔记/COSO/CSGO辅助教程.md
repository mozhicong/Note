# 矩阵

## 1.自己矩阵的特征：4x4的

第一个特征：

```
array[0][0]

- 不开镜的时候小于1
- 第一次开镜会大于1
- 第二次开镜会到5
```

![](CSGO辅助教程.assets/1.jpg)

---

第二个特征：

```
array[3][0]		array[3][1]		array[3][2]
array[4][0]		array[4][1]		array[4][2]
(相同)		   (相同)			  (相同)
```
